PPOV Extractor 系統更新日誌

v1.8.0 - 精簡權限控管與公用備份版 2026-05-28
  • 核心免密碼品質檢查員模式：預設為品檢員角色，擁有載入、輸入、及匯出規格單權限。
  • 新增管理員密碼驗證：僅系統管理員需密碼登入即可解鎖全部 PDF 提取與 CRUD 修改權限。
  • 新增伺服器端公用備份：系統管理員導出總表時，自動同步備份一份至設定權限之公用資料夾內。
v1.7.1 - 管理員密碼自助修改版 2026-05-27
  • 新增密碼自助修改功能：允許管理員透過 UI 修改密碼，並即時持久化保存。
  • 新增安全邊界設計：密碼長度規範（≥6 字元）與舊密碼驗證。
  • 優化修改密碼 Modal：含 Shake 抖動錯誤動畫與成功自動關閉提示。
v1.7.0 - 修改密碼 UI 整合版 2026-05-27
  • 新增在 User Profile 區新增藍色鑰匙按鈕，引導至密碼修改介面。
  • 優化前端 app.js 整合修改密碼端點監聽與非同步提交邏輯。
v1.6.0 - 單檔 PDF 自動化導入版 2026-05-27
  • 新增「新增品號」對接單一 PDF 解析引擎，實現上傳即解析功能。
  • 新增後端 /api/db/import_pdf 端點，支援單檔解析與庫內增量合併。
  • 優化導入成功後自動搜尋、高亮、選中該品號並即時預覽。
v1.5.0 - 資料庫管理後台與 Morandi 視覺版 2026-05-27
  • 核心將工具升級為「規格數據庫後台管理面板」，支援完整 CRUD。
  • 新增冰藍莫蘭迪玻璃卡片設計，頂部配備 4 張動態統計看板。
  • 優化大數據渲染效能，採用 Event Delegation 與 DocumentFragment 防凍結。
  • 修正匯入資料即時持久化存檔，防止刷新後數據遺失。
v1.4.0 - 欄位排序與遞迴子資料夾處理版 2026-05-27
  • 新增彙總表欄位排序功能，點擊表頭即可按產品型號、名稱、版次、模具編號、機台編號排序。
  • 新增排序圖示（向上/向下箭頭），直覺顯示目前排序方向。
  • 新增後端支援遞迴處理多層子資料夾，自動搜尋所有子資料夾中的 PDF 檔案。
  • 優化排序時保留搜尋篩選結果，兩功能完美結合。
v1.3.0 - 查檢數據即時輸入與 AI 匯入預備版 2026-05-27
  • 新增「輸入查檢數據」按鈕，支援即時在網頁端鍵盤輸入實際值、查檢值與簽名欄位。
  • 新增輸入模式切換動畫（Material Design 底線輸入框、Enter 鍵快捷跳轉、自動聚焦）。
  • 預備AI 照片匯入按鈕預留（Coming Soon tooltip 提示）。
  • 優化查檢數據於匯出 Excel 時自動寫入實際值 / 查檢值 / 簽名區域。
v1.2.0 - 冰藍極致視覺與免伺服器雙引擎版 2026-05-27
  • 優化全面導入 MeetingSummary Pro 冰藍莫蘭迪淺色主題，全站符合 WCAG AA/AAA 閱讀對比規範。
  • 新增前端免伺服器雙引擎（SheetJS 讀取總表 + ExcelJS 生成規格單），支援靜態託管部署。
  • 清爽移除冗餘 Git 倉儲卡片，介面專注核心工作流。
v1.1.0 - 多維參數提取與雙處理緒防凍結版 2026-05-25
  • 新增新增模溫（母模/公模/滑塊）、週期、鎖模力、模重等 14+ 組關鍵成型規格參數提取。
  • 修正獨立子進程啟動 TKinter 對話框，徹底解決 Flask Request 執行緒 GUI 當機與掛起問題。
  • 修正單目標參考參數雙向探測定位，完美克服跨行錯位與交叉匹配。
v1.0.0 - 核心解析發布版 2026-05-20
  • 核心射出成型 PDF 報告製程規格單自動化提取引擎。
  • 核心基於 openpyxl 的精美 Excel 查檢表生成與輸出功能。
