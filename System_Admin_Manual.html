PPOV Extractor 系統管理員手冊

版本：v1.8.2 | 建立日期：2026-05-28

1. 系統概述

PPOV Extractor 是一款專為模具參數提取設計的自動化工具。它能從 PPOV PDF 報告中識別關鍵製程參數（如壓力、溫度、時間等），並將其結構化匯出為 Excel 或 JSON 格式。

2. 部署與安裝

2.1 單一執行檔部署

將 dist/PPOV-Extractor.exe 複製到目標伺服器或工作站的任何目錄中。程式執行時會自動在同目錄下生成必要的設定檔。

2.2 資料持久化架構

系統採用「資源與資料分離」設計：

  • 資源 (唯讀)： 包含在 .exe 內的靜態 UI 與模板，由 PyInstaller 管理。
  • 資料 (持久化)： 儲存於 .exe 所在的目錄。
管理員修改的密碼與提取的資料庫檔案將儲存在 .exe 旁邊，請勿隨意移動 users.json。

3. 帳號與安全管理

3.1 預設帳號

用戶名初始密碼權限等級
adminADMIN (大寫)系統管理員

3.2 密碼修改流程

1. 登入系統後點擊「管理員中心」。
2. 選擇「修改密碼」。
3. 變更後將即時加密儲存於 users.json。

隱私風險提示： users.json 儲存了密碼的 SHA-256 雜湊值。請確保該檔案不被非授權人員存取。

4. 欄位設定 (config.json)

如需調整 PDF 提取邏輯，請編輯 config.json。該檔案支援以下配置：

  • fields_to_extract: 定義欄位名稱與正則表達式關鍵字。
  • public_export_folder: 指定匯出 Excel 的預設伺服器路徑。

5. 維護與故障排除

5.1 檔案清單

  • PPOV-Extractor.exe: 主程式
  • users.json: 帳戶資料
  • ppov_database.json: 本地 Master Table 紀錄
  • config.json: 欄位識別設定

5.2 Windows 封鎖問題

若遇到 SmartScreen 警告，請在檔案上點擊 右鍵 -> 內容 -> 勾選「解除封鎖」。
